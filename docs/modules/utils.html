parseKey

  • parseKey(key: Buffer): KeyPath
    • Defined in src/utils.ts:121

    Converts key buffer back into KeyPath e.g. !A!!B!C => ['A', 'B', 'C'] (where ! is the sep) Returned parts are always buffers

    BNF grammar of key buffer: path => levels:ls keyActual:k -> [...ls, k] | keyActual:k -> [k] levels => level:l levels:ls -> [l, ...ls] | '' -> [] level => sep .?:l (?<!escape) sep (?>.) -> l sep => 0x00 escape => 0x5c keyActual => .*:k -> [k]

    Parameters

    • key: Buffer

    Returns KeyPath
